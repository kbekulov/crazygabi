# Gabi Character Direction

Gabi is the player character and should consistently read as:

- Long brown hair
- Fair skin
- Stylish, put-together outfit
- Bright retro platformer proportions: chunky chibi silhouette, strong dark outlines, readable red outfit, clear jump/walk/hurt poses

When replacing or editing character art, keep those traits intact. The current implementation generates original in-game Gabi frames at runtime, inspired by 1990s platformer sprite sheets but not copied from them.
